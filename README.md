# TestNodeAlkemy

## clone project
```
git clone https://github.com/EliasMoura2/TestNodeAlkemy.git
```
## Install dependecies
```
cd TestNodeAlkemy && npm i
```
## Run project
```
npm run dev
```
## Run seeds
```
npx sequelize-cli db:seed:all
```
## Request Postman
```
http://localhost:5000/api/
```
